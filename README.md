# servo-skeleton

reanimation of the dead

arduino code to control continous servos attached to Halloween props, such as this skeleton:

![alt text](https://raw.githubusercontent.com/paidforby/servo-skeleton/master/skeleton.jpg "spooky-servo-skeleton")

hardware setup:
* custom roof mount, with pulleys and strings down to skeleton
* three continous servos (2 FeeTech FS5103R, 1 FS90R)
* Arduino Uno
![alt text](https://raw.githubusercontent.com/paidforby/servo-skeleton/master/servo.jpg "not-so-spooky-servo-mount")
